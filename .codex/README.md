# Orc Camp Codex Setup

This directory contains project-scoped Codex configuration for Orc Camp.

- `agents/`: product design and architecture subagents copied from OrcCamp repository.
- `config.toml`: PixelLab MCP server definition. The real PixelLab bearer token must be provided through the `AUTH_HEADER` environment variable and should not be committed.
- `SUBAGENTS.md`: product subagent index for this repository.
